# Phishing URL Detection System

Machine learning-based URL phishing detection using Random Forest classifier.

## Features

- URL structural analysis for phishing indicators
- HTML content inspection and pattern detection
- Shannon entropy calculation for randomness detection
- Homograph attack pattern recognition
- Port and protocol validation
- Keyword-based malicious URL detection

## Installation

```bash
pip install -r requirements.txt
```

## Usage

Run the Streamlit application:

```bash
streamlit run randomforest.py
```

The application requires a pre-trained model file `phishing_model.pkl`. Train the model first using `train.py`.

## Model Performance

The Random Forest classifier achieves high accuracy in distinguishing between legitimate and phishing URLs through analysis of 67+ features derived from URL structure and HTML content.

## Architecture

### URL Features
- Character distribution and special character analysis
- Domain and subdomain analysis
- Path length and complexity metrics
- HTTPS validation
- IP address detection

### HTML Features
- Form analysis and submission validation
- Embedded brand name detection
- External resource URL tracking
- JavaScript execution patterns
- Favicon external reference detection

### Advanced Detection
- Port anomaly analysis
- Vowel-consonant ratio analysis for homograph attacks
- Consecutive digit sequence detection
- Suspicious keyword scoring

## File Structure

- `randomforest.py` - Main Streamlit application
- `phishing_model.pkl` - Pre-trained model (generated by train.py)
- `requirements.txt` - Python dependencies
- `.gitignore` - Git ignore patterns

## Technical Details

The system analyzes URLs at multiple levels:

1. Protocol validation (HTTP vs HTTPS)
2. Domain component extraction and analysis
3. Subdomain level inspection
4. Path structure evaluation
5. Query parameter analysis
6. HTML content examination when accessible

## Notes

- HTML analysis requires internet connectivity to fetch and analyze web pages
- Page fetching uses a 10-second timeout to prevent hanging on unresponsive servers
- Model predictions include confidence scores for transparency
- Default values are used for HTML features when pages cannot be accessed

- Python 3.6+
